# Chronic Disease Risk Analysis using NHANES Dataset

# Project Overview
This project analyzes health and demographic data from the National Health and Nutrition Examination Survey (NHANES) to identify key factors associated with chronic disease risk among adults in the United States. The goal of the analysis is to explore relationships between clinical measurements, lifestyle factors, and disease outcomes, and to evaluate the effectiveness of machine learning models in predicting potential health risks.
The project demonstrates an end-to-end data science workflow including data cleaning, exploratory analysis, feature engineering, statistical modeling, and machine learning evaluation.

# Dataset
The dataset used in this project comes from the CDC NHANES (National Health and Nutrition Examination Survey).
NHANES is a large-scale public health dataset that includes:
Demographic information
Laboratory test results
Physical examination measurements
Health and lifestyle questionnaire data

For this analysis, multiple NHANES datasets were merged using the participant identifier to create a unified dataset containing key health indicators.

Example variables used in the analysis include:
Age, BMI, Blood pressure, Total cholesterol, HbA1c levels, Smoking status, and Diabetes diagnosis indicators

# Objectives
The main objectives of this project are:
Analyze health and lifestyle variables associated with chronic disease risk.
Perform exploratory data analysis to understand relationships between key variables.
Build predictive models to classify individuals at risk of chronic diseases.
Identify the most influential health indicators affecting disease outcomes.

# Data Processing
The analysis involved several preprocessing steps:
Cleaning and filtering incomplete or missing records.
Merging multiple NHANES datasets using the participant ID.
Feature engineering for derived health indicators.
Standardizing and preparing variables for statistical modeling.

# Methods and Models
Multiple statistical and machine learning models were applied to evaluate chronic disease risk.
Models used include:Logistic Regression, K-Nearest Neighbors, Decision Trees, and Random Forest.
These models were evaluated based on classification accuracy and their ability to identify key health predictors.

# Exploratory Data Analysis
Exploratory analysis was performed to understand patterns and relationships in the data.
Examples of analysis include:
Distribution of BMI and blood pressure across age groups
Correlation analysis between metabolic health indicators
Visualization of risk factors associated with diabetes and cardiovascular conditions

# Key tools used for visualization
Matplotlib, and Seaborn
